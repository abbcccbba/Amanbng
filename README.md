## 立即获取体验新世界！

*其实跟上面一样，软件的获取方法会因为更新而变动*

*也实际上发布页面的安装方法已经写的很详细了，但可能还是有人看不懂，所以在这里补充一下*

<br>

这段介绍的是使用安装程序的安装方法（*如果安装方法不会更改的话*）

1.安装证书 （证书通用，之前的证书也可以）

- 双击证书
- 点击”安装证书“
- 将”存储位置“更改为”本地计算机“
- 选择”将所有证书都放入下列存储“
- （存储路径是”受信任的根证书颁发机构“
- 然后一直下一步就可以了
  
2.双击安装包文件安装
  
- 你可以点击”安装“进行安装

- 安装完毕后你可以点击”打开“ 来打开安装好的应用

- 你可以在开始菜单查看已安装的应用

<br>
### 安装失败？
- 你的系统不是win10以下下
- 你的系统版本处于win10早期
- 你的系统没有powershell
- 你的系统没有Microsoft Store
- 你的系统没有应用安装器（程序）
